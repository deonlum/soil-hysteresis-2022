# Hysteresis in soil microbiome recovery following severe drought

This repository contains R scripts for running statistical analysis and (re)producing figures (in the code folder). Manuscript figures are available in the figures folder. Code for replicating these figures using set seeds and a select number of replicates are available. In some cases, these sections of code are commented out and replaced with a lower number of reps/iterations for quicker analysis that produces similar results.
